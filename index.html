Kalender-Abonnements

Diese Seite bietet herunterladbare .ics-Kalenderdateien für bestimmte Regionen oder Kontexte an. Die Links können in gängigen Kalenderanwendungen importiert werden, sofern diese das iCalendar-Format unterstützen.

⚠️ Nutzung auf eigene Gefahr. Diese Dateien werden ohne jegliche Gewähr bereitgestellt — insbesondere nicht für Richtigkeit, Vollständigkeit, Verfügbarkeit oder Relevanz.

Die Kalenderdateien enthalten Ereignisse von drei Jahren in der Vergangenheit bis fünf Jahren in die Zukunft.

Dieses Projekt folgt einem Rolling-Release-Modell und wird kontinuierlich aktualisiert.

Beiträge sind willkommen! Falls Du einen Fehler findest, erstelle gerne ein Issue oder reiche direkt einen Fix via GitHub ein.
